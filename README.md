# MGSuite-2020
---

*MGSuite-2020* è un progetto per la creazione e la gestione delle e-mail istituzionali degli studenti del G. Marconi.

Il programma verrà scritto nel linguaggio di programmazione C# usando il FrameWork .NET Core 3.1 WPF


## Come funziona
---
Il programma permetterà la creazione automatica delle e-mail e la successiva gestione.

I dati anagrafici degli studenti saranno presi dalla anagrafica di Spaggiari.
